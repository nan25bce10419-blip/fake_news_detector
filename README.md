📌Project Overview

This project focuses on detecting whether a news article is real or fake using machine learning techniques. It uses Natural Language Processing (NLP) to analyze textual data and classify it accordingly.

----

🚀 Features
Data preprocessing and cleaning
TF-IDF based feature extraction
Logistic Regression model
Accuracy evaluation
Command-line execution

----

🎯 Objective

To build a model that can identify fake news accurately
To apply NLP techniques for text processing
To analyze and classify news articles effectively

----

📂 Dataset

The dataset contains news articles labeled as real or fake.
It includes features such as:
Text (news content)
Subject (category)
Target (label: real/fake)

Data Cleaning & Preprocessing
